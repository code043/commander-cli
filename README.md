# Todo CLI

CLI made with CommanderJs

## initializing

### Requirements:

- Vscode
- Node
- Git

### Run the project:

    git clone https://github.com/code043/commander-cli.git

###

    cd commander-cli

###

    npm install

###

    npm link
###

    todo --help
